## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

--Node 
--Python
--Npm 
--Composer 
--Text editor (VsCode)

### Install

1. Clone the repo
   ```sh
   git clone git@github.com:gioudi/repository.git
   ```
2. Install Live Server for frontend
   ```sh
   npm install -g live-server
   ```

```sh
py manage.py runserver
```
